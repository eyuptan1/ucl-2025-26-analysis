# UEFA Champions League 2025-26 Veri Analizi

## Proje Hakkında
Bu proje, UEFA Şampiyonlar Ligi 2025-26 sezonu maç verilerini
kullanarak çeşitli istatistiksel analizler ve makine öğrenmesi
modeli geliştirmeyi amaçlamaktadır.

## Öğrenci Bilgileri
- **Ad Soyad:** Eyüp Tan
- **Öğrenci No:** 1306230003

## Veri Kaynağı
- **Kaynak:** Footiqo.com
- **Lisans:** Ücretsiz, kamuya açık veri
- **Veri Seti:** UCL 2025-26 maç istatistikleri (189 maç)
- **Dosyalar:** overview.csv, scores.csv, corners.csv, attack.csv, xg.csv

## Araştırma Soruları
1. Takımları şanslılıklarına göre sırala — xG ile gerçek gol farkı en yüksek ve en düşük takımlar hangileri?
2. Hangi hakem daha güzel futbol oylatıyor? Hakem bazında ortalama gol, şut ve sarı kart analizi
3. UCL 2025-26'da ikinci yarıda mı daha fazla gol atıldı?
4. İlk yarıda öne geçen takımların maç sonu galibiyet oranı nedir?

## Kullanılan Kütüphaneler

- matplotlib
- seaborn
- scikit-learn
- scipy

1. Repoyu klonla: `git clone https://github.com/eyuptan1/ucl-2025-26-analysis.git`
